## Funksiyalar

### 🔐 Autentifikatsiya
- Login/Logout + Role-based access (Admin / O'qituvchi / Xodim)
- Nofaol hisoblar tizimga kira olmaydi

### 👥 Xodimlar boshqaruvi (faqat Admin)
- Xodim qo'shish, tahrirlash, o'chirish
- Faollashtirish / O'chirish (toggle)
- Rol belgilash: Admin, O'qituvchi, Xodim

### 📚 Kurslar — CRUD, offline/online, narx, jadval
### 🎓 Talabalar — CRUD, QR ID, to'lov tarixi, modal
### 👨‍🏫 O'qituvchilar — kurslar, daromad statistikasi
### 💳 Moliya — to'lovlar, tasdiqlash, CSV eksport
### 💸 Rasxodlar (Admin + Xodim)
- Kategoriyalar: Ijara, Kommunal, Ish haqi, Jihozlar, Marketing...
- Kategoriya bo'yicha grafik
- CSV eksport

### 🏆 Gamifikatsiya (hamma uchun)
- **Reyting sahifasi**: podium (top 3) + to'liq reyting
- **Darajalar**: Yangi boshlovchi → Izlanuvchan → Faol → Yulduz → Chempion
- **Bal berish** (Admin/O'qituvchi/Xodim): sabab, miqdor, tezkor sabablar
- **Sovg'alar**: kerakli bal, ombor, taqdim etish
- **Redeem**: talabani tanlash → baldan ayirish → sovg'a berilishi
